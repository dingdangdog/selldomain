## GetStarted

1. Modify the environment variable value in the [.env](./.env) file.

```sh
VITE_DOMAIN=AitLog.COM
VITE_EMAIL=contact@aitlog.com
```

2. Run `npm run build-only` in Command Line.
3. Deploy Your `dist`.
